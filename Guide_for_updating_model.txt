SETUP
a. Make sure MySQL server is running with instance localhost
b. Make sure localhost instance has username root and password takecare
c. Make sure the latest citybike_test database schema is in database ('~/database_test_create.sql')

ACTIONS
1. In Visual Studio, open '~/CityBikes/Library/GeneratedDatabaseModel/Model.edmx'
2. Select all entities and delete them
3. Right-click on background, click 'Update Model from Database'
4. Open the 'Add' tab
  4.a. Check 'Tables'
  4.b. Check 'Pluralize or singularize generated object names'
5. Click 'Finish'
